# 11055번 - 가장 큰 증가하는 부분 수열

**난이도: Silver 2**

## 문제
수열 A가 주어졌을 때, 그 수열의 증가하는 부분 수열 중에서 합이 가장 큰 것을 구하는 프로그램을 작성하시오.
예를 들어, 수열 A = {1, 100, 2, 50, 60, 3, 5, 6, 7, 8} 인 경우에 합이 가장 큰 증가하는 부분 수열은 A = {1, 100, 2, 50, 60, 3, 5, 6, 7, 8} 이고, 합은 113이다.

## 입력
첫째 줄에 수열 A의 크기 N (1 ≤ N ≤ 1,000)이 주어진다.
둘째 줄에는 수열 A를 이루고 있는 Ai가 주어진다. (1 ≤ Ai ≤ 1,000)

## 출력
첫째 줄에 수열 A의 합이 가장 큰 증가하는 부분 수열의 합을 출력한다.

## 예제 입력 1
```
10
1 100 2 50 60 3 5 6 7 8
```

## 예제 출력 1
```
113
```

---

## 소스코드 (C#)
```cs
namespace ConsoleApp3
{
    internal class Program
    {

        static void Main(string[] args)
        {
            //10개 중에서 수열중에서 가장 큰 수열은?
            int a = int.Parse(Console.ReadLine());
            int[] nums = new int[1000 + 1];
            int[] v = Array.ConvertAll(Console.ReadLine().Split(), int.Parse);

            //수열들을 넣어두거 만약 이전 수보다 더 큰 수가 들어오면 바꿔주자.
            for (int i = 0; i < a; i++)
            {
                int max = 0;
                //j전까지 구할건데. j보다는 작지만 나머지 중에서 가장 큰수를 넣어.
                for (int j = 0; j < i; j++)
                {
                    if (v[i] > v[j] &&max < nums[j] && nums[i] < nums[j])
                    {
                        max = nums[j];
                    }


                   // Console.WriteLine(max + " " + nums[i] + " " + nums[j]);

                }
                //합한 max를 i배열에 넣어준다.
                nums[i] = v[i] + max;
            }
            //이중 for문 돌면서 이전 녀석들 중에서 가장 큰 수를 max로 해서 끝나고 재귀로 
            Console.WriteLine(nums.Max());
        }


    }
    //7 3 8 7 5
}

```
