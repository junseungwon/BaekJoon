# 7562번 - 나이트의 이동

**난이도: Silver 1**

## 문제
체스판 위에 한 나이트가 놓여져 있다. 나이트가 한 번에 이동할 수 있는 칸은 아래 그림에 나와있다. 나이트가 이동하려고 하는 칸이 주어진다. 나이트는 몇 번 움직이면 이 칸으로 이동할 수 있을까?

## 입력
입력의 첫째 줄에는 테스트 케이스의 개수가 주어진다.
각 테스트 케이스는 세 줄로 이루어져 있다. 첫째 줄에는 체스판의 한 변의 길이 l(4 ≤ l ≤ 300)이 주어진다. 체스판의 크기는 l × l이다. 체스판의 각 칸은 두 수의 쌍 {0, ..., l-1} × {0, ..., l-1}로 나타낼 수 있다. 둘째 줄과 셋째 줄에는 나이트가 현재 있는 칸, 나이트가 이동하려고 하는 칸이 주어진다.

## 출력
각 테스트 케이스마다 나이트가 최소 몇 번만에 이동할 수 있는지 출력한다.

## 예제 입력 1
```
3
8
0 0
7 0
100
0 0
30 50
10
1 1
1 1
```

## 예제 출력 1
```
5
28
0
```

## 소스코드 (C#)
```cs
namespace ConsoleApp3
{
    internal class Program
    {
        public static int[] dx = { 1, 2, 2, 1, -1, -2, -2, -1 };
        public static int[] dy = { 2, 1, -1, -2, -2, -1, 1, 2 };
        public static int maxCnt = 0;
        public static Queue<(int y, int x)> queue = new Queue<(int y, int x)>();

        public static void Main()
        {
            int a = int.Parse(Console.ReadLine());
            for (int i = 0; i < a; i++)
            {
                int I = int.Parse(Console.ReadLine());
                int[,] arr = new int[I, I]; // 실제 사용하지 않지만 크기 기준
                bool[,] visited = new bool[I, I];

                int[] start = Array.ConvertAll(Console.ReadLine().Split(), int.Parse);
                int[] finish = Array.ConvertAll(Console.ReadLine().Split(), int.Parse);

                BFS((start[1], start[0]), (finish[1], finish[0]), I, visited);
                Console.WriteLine(maxCnt);
            }
        }

        public static void BFS((int y, int x) startpos, (int y, int x) finishPos, int size, bool[,] visited)
        {
            maxCnt = 0;
            queue.Clear();
            if (startpos == finishPos)
            {
                return;
            }

            queue.Enqueue(startpos);
            visited[startpos.y, startpos.x] = true;
            int level = 0;

            while (queue.Count > 0)
            {
                int qSize = queue.Count;
                for (int i = 0; i < qSize; i++)
                {
                    var pos = queue.Dequeue();

                    for (int dir = 0; dir < 8; dir++)
                    {
                        int nx = pos.x + dx[dir];
                        int ny = pos.y + dy[dir];

                        if (nx >= 0 && nx < size && ny >= 0 && ny < size && !visited[ny, nx])
                        {
                            if (nx == finishPos.x && ny == finishPos.y)
                            {
                                maxCnt = level + 1;
                                return;
                            }

                            visited[ny, nx] = true;
                            queue.Enqueue((ny, nx));
                        }
                    }
                }
                level++;
            }
        }
    }
}

```
