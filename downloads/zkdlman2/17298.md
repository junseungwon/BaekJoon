# 17298번 - 오큰수

**난이도: Gold 4**

## 문제
크기가 N인 수열 A = A1, A2, ..., AN이 있다. 수열의 각 원소 Ai에 대해서 오큰수 NGE(i)를 구하려고 한다. Ai의 오큰수는 오른쪽에 있으면서 Ai보다 큰 수 중에서 가장 왼쪽에 있는 수를 의미한다. 그러한 수가 없는 경우에 오큰수는 -1이다.
예를 들어, A = [3, 5, 2, 7]인 경우 NGE(1) = 5, NGE(2) = 7, NGE(3) = 7, NGE(4) = -1이다. A = [9, 5, 4, 8]인 경우에는 NGE(1) = -1, NGE(2) = 8, NGE(3) = 8, NGE(4) = -1이다.

## 입력
첫째 줄에 수열 A의 크기 N (1 ≤ N ≤ 1,000,000)이 주어진다. 둘째 줄에 수열 A의 원소 A1, A2, ..., AN (1 ≤ Ai ≤ 1,000,000)이 주어진다.

## 출력
총 N개의 수 NGE(1), NGE(2), ..., NGE(N)을 공백으로 구분해 출력한다.

## 예제 입력 1
```
4
3 5 2 7
```

## 예제 출력 1
```
5 7 7 -1
```

## 예제 입력 2
```
4
9 5 4 8
```

## 예제 출력 2
```
-1 8 8 -1
```

---

## 소스코드 (C#)
```cs
using System.Text;
namespace CodeTesting
{
    internal class Program
    {

        static void Main(string[] args)
        {
            //오큰수를 구함 크기가 n이다.

            //A의 i번째는 Ai의 오큰수는 오른쪽에 있으면서 Ai보다 큰 수 중에서 가장 왼쪽에 있는 수를 의미한다. 없으면 -1이다.
            //NGE(Ai) = 오른쪽에 있으면서 Ai보다 큰 수 중에서 가장 왼쪽에 있는 수
            //NGE(1) = 오른쪽에 있으면서 5보다 큰 수 중에서 가장 왼쪽에 있는 수

            //뒤에 있는 수 중에서 ai보다 크지만 최대한 왼쪽에 있는 수

            //왼쪽에 있는 수를 기준으로 왼쪽의 수보다 크고 가장 빠른 수
            //1을 입력하면 0번째의 수보다 크면서 가장 빠르게 나오는 수를 구해라

            Stack<int> stack = new Stack<int>();
            StringBuilder sb = new StringBuilder();

            int num = int.Parse(Console.ReadLine());
            //각 숫자를 받은 문자열 배열들
            string[] arr1 = Console.ReadLine().Split();
            int[] s = Array.ConvertAll(arr1, int.Parse);
            //배열은 0부터 시작인데 num은 +1부터라 문제가 생김

            for (int i = 0; i < num; i++)
            {
                //배열 i번째 수와 stack수를 비교함
                while (stack.Count > 0 && s[stack.Peek()] < (s[i]))
                {
                    int pop = stack.Pop();

                    s[pop] = s[i];

                }


                stack.Push(i);
            }
            //아직도 stack에 남아있는 수를 모두 -1로 표기
            while (stack.Count > 0)
            {
                s[stack.Pop()] = -1;
            }

            for (int i = 0; i < s.Length; i++)
            {
                sb.Append(s[i]+" ");
            }
            Console.WriteLine(sb);
        }

    }

}


```
