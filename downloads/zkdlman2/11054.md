# 11054번 - 가장 긴 바이토닉 부분 수열

**난이도: Gold 4**

## 문제
수열 S가 어떤 수 Sk를 기준으로 S1 < S2 < ... Sk-1 < Sk > Sk+1 > ... SN-1 > SN을 만족한다면, 그 수열을 바이토닉 수열이라고 한다.
예를 들어, {10, 20, 30, 25, 20}과 {10, 20, 30, 40}, {50, 40, 25, 10} 은 바이토닉 수열이지만,  {1, 2, 3, 2, 1, 2, 3, 2, 1}과 {10, 20, 30, 40, 20, 30} 은 바이토닉 수열이 아니다.
수열 A가 주어졌을 때, 그 수열의 부분 수열 중 바이토닉 수열이면서 가장 긴 수열의 길이를 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 수열 A의 크기 N이 주어지고, 둘째 줄에는 수열 A를 이루고 있는 Ai가 주어진다. (1 ≤ N ≤ 1,000, 1 ≤ Ai ≤ 1,000)

## 출력
첫째 줄에 수열 A의 부분 수열 중에서 가장 긴 바이토닉 수열의 길이를 출력한다.

## 예제 입력 1
```
10
1 5 2 1 4 3 4 5 2 1
```

## 예제 출력 1
```
7
```

---

## 소스코드 (C#)
```cs
namespace ConsoleApp3
{
    internal class Program
    {

        static void Main(string[] args)
        {
            //sk 가운데를 기준으로 + -가 되는 수열을 구해라

            //가장 긴 바이토닉 수열의 길이를 구해라 수를 구할 때
            //이차원 배열로 +가 0 -가 1로 정한다.
            //본인 이전의 수 중에서 가장+값이 가장 긴 값을 저장한다.

            //연속하는 수열이지.
            //증가하는 부분은 i-1에서 시작해서 이전값에 증가하는 길이를 가져올거임 만약 v[i] > v[j]라면

            //최종적으로는 +- +1을 한 것이 가장 긴 바이토닉 수열이 될거다.
            int a = int.Parse(Console.ReadLine());
            int[] nums = Array.ConvertAll(Console.ReadLine().Split(), int.Parse);

            int[,] v = new int[a, 2];
            v[0, 0] = 0; v[a - 1, 1] = 0;
            for (int i = 1; i < a; i++)
            {

                //증가하는 길이를 저장한다.(왼쪽 값)
                for (int j = 0; j < i; j++)
                {
                    if (nums[i] > nums[j])
                    {
                        v[i, 0] = Math.Max(v[i,0], v[j,0]+1);
                    }
                }
            }

            for (int i = a - 2; i >= 0; i--)
            {
                for (int j = a - 1; j > i; j--) {
                    if (nums[i] > nums[j])
                    {
                        v[i, 1] = Math.Max(v[i, 1], v[j, 1] + 1);
                    }
                }
            }
            
            int max = 0;
            for(int i=0; i<a; i++)
            {
                max = (max < v[i, 0] + v[i, 1] + 1) ? v[i, 0] + v[i, 1] + 1 : max;
            }
            Console.WriteLine(max);
        }

    }
}

```
