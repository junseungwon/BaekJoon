# 15990번 - 1, 2, 3 더하기 5

**난이도: Silver 1**

## 문제
정수 4를 1, 2, 3의 합으로 나타내는 방법은 총 3가지가 있다. 합을 나타낼 때는 수를 1개 이상 사용해야 한다. 단, 같은 수를 두 번 이상 연속해서 사용하면 안 된다.

1+2+1
1+3
3+1

정수 n이 주어졌을 때, n을 1, 2, 3의 합으로 나타내는 방법의 수를 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 정수 n이 주어진다. n은 양수이며 100,000보다 작거나 같다.

## 출력
각 테스트 케이스마다, n을 1, 2, 3의 합으로 나타내는 방법의 수를 1,000,000,009로 나눈 나머지를 출력한다.

## 예제 입력 1
```
3
4
7
10
```

## 예제 출력 1
```
3
9
27
```

---

## 소스코드 (C#)
```cs
using System.Text;

namespace CodeTesting
{
    internal class Program
    {
        static void Main(string[] args)
        {

            //연속으로 사용하지말고 숫자들을 모두 한번씩 사용해야한다.

            //2 1+1 2
            //3 1+1+1 2+1 1+2 3 ->2+1 1+2 3 ->3
            //4 1+1+1+1 2+2 2+1+1 1+1+2 1+2+1 3+1 1+3 ->7 1+2+1 3+1 1+3 ->4
            // 1 -> 1
            // 2 -> 0
            // 2+1 1+2 3 ->2 
            // 1+2+1 3+1 1+3
            // 1+3+1 2+3 3+2


            //5 1+2+1+1? 1+3+1 4+1 1+4 2+3 3+2
            int max = 100000;
            long ab = 1000000009;
            double[,] nums = new double[max+2, 4];
            nums[1, 1] = 1;
            nums[2, 2] = 1;
            nums[3, 1] = 1;
            nums[3, 2] = 1;
            nums[3, 3] = 1;
            //각 배열에 값을 넣어준다.
            for(int i=4; i<= max+1; i++)
            {
                nums[i, 1] = (nums[i-1,2]+nums[i-1,3])%ab;
                nums[i, 2] = (nums[i-2,1]+nums[i-2,3])%ab;
                nums[i, 3] = (nums[i-3,2]+nums[i-3,1])%ab;
            }

            StringBuilder sb = new StringBuilder();
            int a = int.Parse(Console.ReadLine());
            for(int i=0; i<a; i++)
            {
                int b = int.Parse(Console.ReadLine());
                sb.AppendLine($"{(nums[b,1]+ nums[b, 2]+ nums[b, 3])%1000000009}");

            }
            Console.WriteLine( sb.ToString() );
        }
    }
}

```
