# 1260번 - DFS와 BFS

**난이도: Silver 2**

## 문제
그래프를 DFS로 탐색한 결과와 BFS로 탐색한 결과를 출력하는 프로그램을 작성하시오. 단, 방문할 수 있는 정점이 여러 개인 경우에는 정점 번호가 작은 것을 먼저 방문하고, 더 이상 방문할 수 있는 점이 없는 경우 종료한다. 정점 번호는 1번부터 N번까지이다.

## 입력
첫째 줄에 정점의 개수 N(1 ≤ N ≤ 1,000), 간선의 개수 M(1 ≤ M ≤ 10,000), 탐색을 시작할 정점의 번호 V가 주어진다. 다음 M개의 줄에는 간선이 연결하는 두 정점의 번호가 주어진다. 어떤 두 정점 사이에 여러 개의 간선이 있을 수 있다. 입력으로 주어지는 간선은 양방향이다.

## 출력
첫째 줄에 DFS를 수행한 결과를, 그 다음 줄에는 BFS를 수행한 결과를 출력한다. V부터 방문된 점을 순서대로 출력하면 된다.

## 예제 입력 1
```
4 5 1
1 2
1 3
1 4
2 4
3 4
```

## 예제 출력 1
```
1 2 4 3
1 2 3 4
```

## 예제 입력 2
```
5 5 3
5 4
5 2
1 2
3 4
3 1
```

## 예제 출력 2
```
3 1 2 5 4
3 1 4 2 5
```

## 예제 입력 3
```
1000 1 1000
999 1000
```

## 예제 출력 3
```
1000 999
1000 999
```

## 소스코드 (C#)
```cs
internal class Program
{
    public static bool[] visit;
    static void Main()
    {
        string[] a = Console.ReadLine().Split();
        int x = int.Parse(a[0])+1;
        int y = int.Parse(a[1])+1;
        int start = int.Parse(a[2]);

        int[,] ad = new int[x, x];
        //0부터 입력 받은 값까지니깐 x값

        for(int i=0; i< y-1; i++)
        {
            string[] b = Console.ReadLine().Split();
            ad[int.Parse(b[0]),int.Parse(b[1])] = 1;
            ad[int.Parse(b[1]), int.Parse(b[0])] = 1;
        }

        //PrintArray(x, y, ad);
        visit = new bool[x];
        DFS(start,ad);
        Console.WriteLine();
        visit = new bool[x];
        BFS(start, ad);
    }

    //0을 입력을 하자
    public static void BFS(int startNum, int[,] arr)
    {
        //방문했는지 확인을 위한 bool배열을 생성한다.
        Queue<int> queue = new Queue<int>();
        queue.Enqueue(startNum);
        visit[startNum] = true;

        while(queue.Count > 0)
        {
            int num = queue.Dequeue();
            Console.Write(num+" ");

            for(int i=0; i<arr.GetLength(0); i++)
            {
                //0이면 다음으로 넘어감
                if (arr[num, i] == 0) continue;
                //방문했는지 확인
                if (visit[i] == true) continue;

                queue.Enqueue(i);
                visit[i] = true;
            }
        }
        Console.WriteLine();
    }
    public static void DFS(int startNum, int[,] arr)
    {
        Console.Write(startNum+" ");
        visit[startNum] = true;

        //시작 번호부터 시작해서 해당 arr를 깊이 우선탐색으로 검사한다.
        for (int i = 0; i <arr.GetLength(0); i++)
        {
            //정점이라면 스킵
            if (arr[startNum, i] == 0)
                continue;
            //이미 방문한 곳이면 스킵
            if (visit[i] == true)
                continue;

            DFS(i, arr);
        }
    }

    public static void PrintArray(int x, int y, int[,] arr)
    {
        for(int i=0; i<x; i++)
        {
            for(int j=0; j<y; j++)
            {
                Console.Write(arr[i,j]+" ");
            }
            Console.WriteLine();
        }
    }
    //DFS BFS
    //정렬(버블 정렬, 선택정렬, 삽입 정렬, 병합 정렬, 힙 정렬, 퀵 정렬)
}
```
